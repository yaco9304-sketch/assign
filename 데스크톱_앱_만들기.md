# 데스크톱 앱 만들기 가이드

## 🎯 목표

웹사이트를 **진짜 프로그램(데스크톱 앱)**으로 만들어서:
- ✅ 설치 파일(.exe, .dmg)로 배포
- ✅ 오프라인에서도 완전히 작동
- ✅ 학교 네트워크 문제 없이 사용
- ✅ 네이티브 앱처럼 실행

## 🛠️ 사용 기술

**Electron** - 웹 기술(HTML/CSS/JavaScript)로 데스크톱 앱 만들기

## 📦 설치 방법

### 1단계: 의존성 설치

```bash
cd desktop-app
npm install
```

### 2단계: 개발 모드로 실행

**터미널 1** - 프론트엔드 개발 서버:
```bash
cd frontend
npm run dev
```

**터미널 2** - Electron 앱 실행:
```bash
cd desktop-app
npm run dev
```

이제 데스크톱 앱이 실행됩니다!

## 🏗️ 실행 파일 만들기

### Windows용 (.exe 파일)

```bash
cd desktop-app
npm run build:win
```

생성 위치: `desktop-app/dist/학년 배정 시스템 Setup 1.0.0.exe`

### Mac용 (.dmg 파일)

```bash
cd desktop-app
npm run build:mac
```

생성 위치: `desktop-app/dist/학년 배정 시스템-1.0.0.dmg`

### 모든 플랫폼

```bash
cd desktop-app
npm run build:all
```

## 📤 배포 방법

1. **빌드된 파일 확인**
   - Windows: `dist/학년 배정 시스템 Setup 1.0.0.exe`
   - Mac: `dist/학년 배정 시스템-1.0.0.dmg`

2. **파일 공유**
   - USB에 복사
   - 클라우드 저장소에 업로드
   - 학교 서버에 배포

3. **사용자 설치**
   - Windows: `.exe` 파일 더블클릭 → 설치
   - Mac: `.dmg` 파일 열기 → 앱을 Applications로 드래그

## 🔧 완전한 오프라인 지원

현재는 백엔드 서버가 필요합니다. 완전한 오프라인을 위해서는:

### 옵션 1: 백엔드를 앱에 포함

1. Python 백엔드를 Electron 앱에 포함
2. 앱 실행 시 자동으로 백엔드 서버 시작
3. 완전히 오프라인에서 작동

### 옵션 2: SQLite 로컬 데이터베이스

1. 백엔드 없이 SQLite 직접 사용
2. 프론트엔드에서 직접 데이터베이스 접근
3. 가장 가벼운 방법

## 🎨 아이콘 추가

`desktop-app/assets/` 폴더에 아이콘 추가:

- Windows: `icon.ico` (256x256)
- Mac: `icon.icns` (512x512)
- 공통: `icon.png` (512x512)

아이콘 생성 도구:
- https://www.icoconverter.com/
- https://cloudconvert.com/png-to-ico
