# PWA 쉽게 설명하기

## 🎯 PWA란?

**PWA = 웹사이트를 앱처럼 사용할 수 있게 만드는 기술**

일반 웹사이트와 달리:
- 📱 **앱처럼 설치 가능** (홈 화면에 아이콘 추가)
- ⚡ **더 빠르게 실행** (캐싱 덕분)
- 📴 **오프라인에서도 작동** (인터넷 없어도 일부 기능 사용 가능)

## 🏫 학교에서 왜 안 열리나요?

학교 네트워크에서 웹사이트가 안 열리는 이유:
- 방화벽 설정
- 네트워크 제한
- 보안 정책

## ✅ PWA로 해결하는 방법

PWA를 설치하면:
1. **홈 화면에 아이콘 추가** → 앱처럼 실행
2. **오프라인 지원** → 인터넷 없어도 기본 기능 사용
3. **빠른 로딩** → 캐싱된 데이터로 즉시 실행

## 📱 실제 사용 방법 (단계별)

### Android 폰에서

1. **크롬 브라우저로 웹사이트 접속**
   - 주소창에 `https://assign-nu-weld.vercel.app` 입력

2. **설치 버튼 찾기**
   - 주소창 오른쪽에 "설치" 아이콘 표시
   - 또는 화면 하단에 "홈 화면에 추가" 팝업 표시

3. **설치하기**
   - "설치" 또는 "추가" 버튼 클릭
   - 확인 메시지에서 "설치" 클릭

4. **앱처럼 사용하기**
   - 홈 화면에 "학년배정" 아이콘 생성됨
   - 아이콘 클릭하면 앱처럼 실행됨

### iPhone에서

1. **Safari 브라우저로 웹사이트 접속**
   - 주소창에 `https://assign-nu-weld.vercel.app` 입력

2. **공유 버튼 클릭**
   - 화면 하단 중앙의 "□↑" 아이콘 클릭

3. **홈 화면에 추가**
   - 메뉴에서 "홈 화면에 추가" 선택

4. **이름 확인 후 추가**
   - 이름이 "학년배정"으로 표시됨
   - "추가" 버튼 클릭

5. **앱처럼 사용하기**
   - 홈 화면에 "학년배정" 아이콘 생성됨
   - 아이콘 클릭하면 앱처럼 실행됨

### 컴퓨터에서

1. **크롬/엣지 브라우저로 웹사이트 접속**

2. **주소창 오른쪽의 설치 아이콘 클릭**
   - 또는 메뉴 → "앱 설치" 선택

3. **설치 확인**
   - "설치" 버튼 클릭

4. **앱처럼 사용하기**
   - 바탕화면에 아이콘 생성됨
   - 더블클릭하면 앱처럼 실행됨

## 🔍 설치 확인 방법

설치가 잘 되었는지 확인:
- ✅ 홈 화면에 "학년배정" 아이콘 보임
- ✅ 아이콘 클릭하면 앱처럼 실행됨 (주소창 없음)
- ✅ 인터넷이 없어도 기본 화면은 보임

## 💡 왜 이렇게 하나요?

### 문제 상황
- 학교 네트워크에서 웹사이트 접속 안 됨
- 매번 브라우저에서 주소 입력해야 함
- 인터넷이 없으면 아무것도 못 함

### PWA 설치 후
- ✅ 홈 화면 아이콘으로 바로 실행
- ✅ 오프라인에서도 기본 기능 사용 가능
- ✅ 더 빠르게 실행됨

## 🎨 실제 화면 예시

### 설치 전
```
[브라우저] → 주소 입력 → 웹사이트 열림
```

### 설치 후
```
[홈 화면 아이콘] → 클릭 → 앱처럼 실행
```

## ⚠️ 주의사항

1. **HTTPS 필수**
   - PWA는 HTTPS에서만 작동
   - Vercel 배포 시 자동으로 HTTPS 제공

2. **아이콘 필요**
   - 현재는 기본 아이콘 사용
   - 나중에 커스텀 아이콘 추가 가능

3. **오프라인 기능 제한**
   - 완전한 오프라인은 아님
   - 기본 화면과 캐싱된 데이터만 사용 가능
   - 새 데이터는 인터넷 필요

## 📞 문제 해결

### 설치 버튼이 안 보여요
- 브라우저를 최신 버전으로 업데이트
- HTTPS로 접속했는지 확인
- 다른 브라우저로 시도 (크롬, 엣지, Safari)

### 설치했는데 안 열려요
- 홈 화면 아이콘 확인
- 아이콘을 길게 눌러서 "앱 정보" 확인
- 브라우저 캐시 삭제 후 재설치

### 여전히 학교에서 안 열려요
- PWA 설치 후에도 네트워크 문제는 해결 안 됨
- 하지만 오프라인에서 기본 기능은 사용 가능
- 캐싱된 데이터로 빠르게 로딩됨
