# Vercel 배포 상태 확인 가이드

## 배포 완료 여부 확인 방법

### 1단계: Vercel 대시보드 접속

1. https://vercel.com 접속
2. GitHub 계정으로 로그인 (또는 이미 로그인되어 있으면 스킵)

### 2단계: 프로젝트 선택

1. 대시보드에서 프로젝트 목록 확인
2. `assign` 또는 프로젝트 이름 클릭

### 3단계: Deployments 탭 확인

1. 상단 메뉴에서 **"Deployments"** 탭 클릭
2. 최신 배포 항목이 맨 위에 표시됩니다
3. 각 배포 항목은 카드 형태로 표시되며, **커밋 메시지가 제목처럼 보입니다**

### 4단계: 커밋 정보 확인

배포 항목에서 다음 정보 확인:

**커밋 메시지 보는 방법:**
1. Deployments 탭에서 각 배포 항목(카드)을 확인
2. 배포 항목의 **왼쪽 상단** 또는 **제목 부분**에 커밋 메시지가 표시됨
3. 또는 배포 항목을 클릭하면 상세 페이지에서 확인 가능

**확인할 내용:**
- **커밋 메시지**: `Fix: 학년부장 희망 → 학년부장으로 변경` ✅
- **커밋 해시**: `3673fea` (또는 일부, 보통 메시지 옆에 작게 표시)
- **브랜치**: `main`

**화면 구성 예시:**
```
┌─────────────────────────────────────────┐
│ [초록색 점] Ready                        │
│                                         │
│ Fix: 학년부장 희망 → 학년부장으로 변경   │ ← 여기!
│ main · 3673fea                          │
│                                         │
│ Production                              │
│ 2 minutes ago                           │
└─────────────────────────────────────────┘
```

### 5단계: 배포 상태 확인

배포 항목 옆에 상태 표시가 있습니다:

- ✅ **Ready** (초록색) = 배포 완료, 사이트에서 확인 가능
- ⏳ **Building** (노란색) = 배포 중, 잠시 후 다시 확인
- ❌ **Error** (빨간색) = 배포 실패, Build Logs 확인 필요

### 6단계: 배포 로그 확인 (문제 발생 시)

1. 배포 항목 클릭
2. **"Build Logs"** 탭 클릭
3. 오류 메시지 확인

## 배포가 완료되었는지 확인하는 다른 방법

### 방법 1: 사이트에서 직접 확인

1. https://assign-nu-weld.vercel.app 접속
2. Google 로그인으로 교사 화면 접속
3. Step 1에서 "담임" 체크
4. "학년부장 희망" → "학년부장"으로 변경되었는지 확인

### 방법 2: 브라우저 개발자 도구 확인

1. F12 또는 우클릭 → "검사" 클릭
2. **Network** 탭 열기
3. 페이지 새로고침 (Ctrl+Shift+R 또는 Cmd+Shift+R)
4. JavaScript 파일 이름 확인:
   - 파일명에 해시가 포함되어 있음 (예: `index-abc123.js`)
   - 이전 배포와 해시가 다르면 새 배포가 반영된 것

### 방법 3: 강력 새로고침

브라우저 캐시 문제로 이전 버전이 보일 수 있습니다:

- **Windows/Linux**: `Ctrl + Shift + R`
- **Mac**: `Cmd + Shift + R`
- 또는 시크릿 모드에서 접속

## 문제 해결

### 배포가 "Building" 상태에서 멈춤

- 보통 1-2분 소요
- 5분 이상 걸리면 "Redeploy" 클릭

### 배포가 "Error" 상태

1. Build Logs 확인
2. 오류 메시지 확인
3. 필요시 코드 수정 후 다시 푸시

### 사이트에서 변경사항이 보이지 않음

1. 강력 새로고침 시도
2. 시크릿 모드에서 확인
3. Vercel에서 배포가 완료되었는지 확인
4. 브라우저 캐시 삭제
