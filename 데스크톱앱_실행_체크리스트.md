# 데스크톱 앱 실행 체크리스트

## ✅ 실행 전 확인사항

### 1. 백엔드 서버 실행 (필수!)

**터미널에서 실행:**
```bash
cd backend
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8001
```

**확인 방법:**
```bash
curl http://localhost:8001/docs
```
응답이 오면 정상입니다.

### 2. 앱 실행

1. **Applications에서 실행**
   - Applications 폴더 열기
   - "학년 배정 시스템" 더블클릭

2. **보안 경고 해결**
   - "확인되지 않은 개발자" 경고가 나오면:
   - 시스템 설정 → 보안 및 개인 정보 보호
   - "그래도 열기" 클릭
   - 또는 앱을 우클릭 → "열기"

### 3. 오류 확인

**개발자 도구 열기:**
- Cmd + Option + I
- Console 탭에서 오류 확인

**일반적인 오류:**
- `Failed to fetch` → 백엔드 서버 미실행
- `Network Error` → 백엔드 서버 미실행
- `CORS error` → 백엔드 CORS 설정 확인

## 🔧 문제 해결

### 문제 1: 앱이 열리지 않음

**해결:**
1. 보안 설정 확인
2. 앱을 우클릭 → "열기"
3. 시스템 설정에서 허용

### 문제 2: 로그인 페이지는 보이지만 작동 안 함

**원인:** 백엔드 서버 미실행

**해결:**
```bash
cd backend
python3 -m uvicorn app.main:app --port 8001
```

### 문제 3: "서버에 연결할 수 없습니다" 오류

**원인:** 백엔드 서버가 실행되지 않음

**해결:**
1. 백엔드 서버 실행 확인
2. 포트 8001이 사용 중인지 확인
3. 방화벽 설정 확인

## 🚀 정상 실행 확인

앱이 정상적으로 실행되면:
1. ✅ 로그인 페이지 표시
2. ✅ "Continue with Google" 버튼 표시
3. ✅ 관리자 로그인 가능

## 💡 완전한 해결책

**현재:** 백엔드 서버를 별도로 실행해야 함

**향후 개선:**
- 백엔드를 앱에 포함
- 앱 실행 시 자동으로 백엔드 서버 시작
- 완전히 독립적으로 작동
