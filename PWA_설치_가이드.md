# PWA 설치 가이드

## PWA란?
Progressive Web App (PWA)는 웹 기술로 만든 앱처럼 동작하는 웹 애플리케이션입니다.

## 주요 기능
1. **오프라인 지원**: 인터넷이 없어도 일부 기능 사용 가능
2. **앱 설치**: 홈 화면에 추가하여 앱처럼 사용
3. **빠른 로딩**: 캐싱을 통한 빠른 로딩 속도
4. **푸시 알림**: (향후 추가 가능)

## 설치 방법

### Android (크롬/엣지)
1. 웹사이트 접속
2. 주소창 옆 "설치" 아이콘 클릭
3. 또는 메뉴 → "홈 화면에 추가" 선택
4. "설치" 버튼 클릭

### iOS (Safari)
1. Safari로 웹사이트 접속
2. 하단 공유 버튼(□↑) 클릭
3. "홈 화면에 추가" 선택
4. 이름 확인 후 "추가" 클릭

### 데스크톱 (크롬/엣지)
1. 주소창 오른쪽에 설치 아이콘 표시
2. 클릭하여 설치
3. 또는 메뉴 → "앱 설치" 선택

## 아이콘 생성 필요

PWA 아이콘을 생성해야 합니다:
- `pwa-192x192.png` (192x192 픽셀)
- `pwa-512x512.png` (512x512 픽셀)
- `apple-touch-icon.png` (180x180 픽셀)

이미지 생성 도구:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

## 오프라인 지원

PWA는 다음을 캐싱합니다:
- 정적 파일 (HTML, CSS, JS)
- 이미지 (30일간 캐싱)
- API 응답 (5분간 캐싱, 네트워크 우선)

## 학교 네트워크 문제 해결

PWA를 설치하면:
1. 오프라인에서도 기본 기능 사용 가능
2. 캐싱된 데이터로 빠른 로딩
3. 네트워크 문제 시에도 일부 기능 작동

## 개발 모드

개발 중에는 `vite.config.ts`의 `devOptions.enabled: true`로 설정되어 있어
개발 서버에서도 PWA 기능을 테스트할 수 있습니다.

## 배포 후 확인

1. Vercel에 배포
2. HTTPS로 접속 (PWA는 HTTPS 필수)
3. 브라우저에서 설치 프롬프트 확인
4. 설치 후 홈 화면에서 앱처럼 실행
