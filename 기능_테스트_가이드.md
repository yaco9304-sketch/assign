# 기능 테스트 가이드

## 📋 테스트 목록

배포 완료 후 다음 기능들을 테스트합니다:

1. ✅ 일반 로그인 테스트
2. ✅ Google 로그인 테스트
3. ✅ 관리자 기능 테스트
4. ✅ 교사 기능 테스트

## 🧪 테스트 시나리오

### 1. 일반 로그인 테스트

#### 교사 로그인
1. **사이트 접속**
   - `https://yaco9304-sketch.github.io/assign` 접속

2. **역할 선택**
   - "역할" 드롭다운에서 "교사" 선택

3. **로그인**
   - 비밀번호 입력 (설정된 `TEACHER_PASSWORD`)
   - 로그인 버튼 클릭

4. **확인 사항**
   - ✅ 로그인 성공
   - ✅ "내 희망" 페이지로 이동
   - ✅ 교사 정보 표시

#### 관리자 로그인
1. **역할 선택**
   - "역할" 드롭다운에서 "관리자" 선택

2. **로그인**
   - 비밀번호 입력 (설정된 `ADMIN_PASSWORD`)
   - 로그인 버튼 클릭

3. **확인 사항**
   - ✅ 로그인 성공
   - ✅ 관리자 대시보드로 이동
   - ✅ 통계 정보 표시

### 2. Google 로그인 테스트

1. **사이트 접속**
   - `https://yaco9304-sketch.github.io/assign` 접속

2. **Google 로그인 버튼 클릭**
   - "Continue with Google" 버튼 클릭

3. **Google 계정 선택**
   - Google 계정 선택
   - 권한 동의

4. **확인 사항**
   - ✅ Google 로그인 성공
   - ✅ "내 희망" 페이지로 이동
   - ✅ 교사 정보 자동 생성/업데이트
   - ✅ 토큰 저장 확인

### 3. 관리자 기능 테스트

#### 대시보드 확인
1. **관리자 로그인**
   - 관리자 비밀번호로 로그인

2. **대시보드 확인**
   - ✅ 전체 교사 수 표시
   - ✅ 희망 제출 수 표시
   - ✅ 필요 담임 수 표시
   - ✅ 학년별 통계 표시

#### 희망 제출 마감 기능
1. **"희망 제출 마감" 버튼 클릭**
   - 대시보드에서 버튼 클릭

2. **확인 사항**
   - ✅ 버튼이 "마감 해제"로 변경
   - ✅ 교사 페이지에서 입력 필드 비활성화

3. **"마감 해제" 버튼 클릭**
   - 버튼 클릭

4. **확인 사항**
   - ✅ 버튼이 "희망 제출 마감"으로 변경
   - ✅ 교사 페이지에서 입력 필드 활성화

#### 교사 정보 업로드
1. **"교사 정보 업로드" 섹션**
   - Excel 파일 선택
   - 업로드 버튼 클릭

2. **확인 사항**
   - ✅ 업로드 성공 메시지
   - ✅ 성공/실패 개수 표시
   - ✅ 교사 목록 업데이트

### 4. 교사 기능 테스트

#### 희망 제출
1. **교사 로그인**
   - 교사 비밀번호 또는 Google 로그인

2. **"내 희망" 페이지 확인**
   - ✅ 교사 이름 표시
   - ✅ 학년 선택 드롭다운 표시
   - ✅ 1지망, 2지망, 3지망 입력 필드 표시

3. **희망 제출**
   - 학년 선택
   - 1지망, 2지망, 3지망 입력
   - "제출" 버튼 클릭

4. **확인 사항**
   - ✅ 제출 성공 메시지
   - ✅ 저장된 희망 확인

#### 희망 수정
1. **기존 희망 수정**
   - 학년 또는 지망 변경
   - "제출" 버튼 클릭

2. **확인 사항**
   - ✅ 수정 성공
   - ✅ 변경사항 반영

#### 마감 상태 확인
1. **관리자가 마감한 후**
   - 교사 페이지 접속

2. **확인 사항**
   - ✅ 입력 필드 비활성화
   - ✅ 제출 버튼 대신 "제출이 마감되었습니다" 메시지 표시

## 🐛 문제 발생 시

### CORS 오류
- Render 대시보드에서 `ALLOWED_ORIGINS` 확인
- GitHub Pages URL이 포함되어 있는지 확인
- 서비스 재시작 확인

### 로그인 실패
- 비밀번호 확인
- Google OAuth 설정 확인
- 브라우저 콘솔 오류 확인

### API 연결 실패
- 백엔드 서버 상태 확인
- 네트워크 탭에서 요청 상태 확인
- 백엔드 로그 확인
